# AI Chat Assistant

## Overview
This AI Chat Assistant is designed to provide interactive and intelligent responses to user queries using natural language processing techniques. It leverages a vast dataset to ensure relevant and accurate replies.

## Features
- **Natural Language Understanding:** Understands and responds to user queries in a conversational manner.
- **Multiple Intent Handling:** Capable of understanding multiple intents in a single query.
- **Customizable Responses:** Allows for personalization of responses based on user preferences.

## Installation
To install the AI Chat Assistant:
1. Clone the repository:
   ```bash
   git clone https://github.com/Manas2904/ai-chat-assistant.git
   ```
2. Navigate into the project directory:
   ```bash
   cd ai-chat-assistant
   ```
3. Install the necessary dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
To run the assistant:
```bash
python main.py
```

## Configuration
You can configure the assistant's behavior by modifying the `config.yaml` file. This file includes settings for language, response behavior, and more.
